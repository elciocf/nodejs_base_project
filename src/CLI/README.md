# Para utilizar o CLI de criação de entidades. Informe o JSON com os campos da entidade, editando o arquivo fields.json dentro da pasta CLI


## Exemplo:
`
{
    "cod_token": "string",    
    "cod_usuario": "string",
    "refresh_token": "string",
    "expira_em": "Date",
    "created_at?": "Date",
    "updated_at?": "Date"
}
`
